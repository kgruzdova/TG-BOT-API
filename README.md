# Telegram-бот с OpenAI (aiogram 3)

Telegram-бот с поддержкой памяти диалога, переключением режимов, генерацией видео (Sora 2) и интеграцией OpenAI API.

**Требования:** Python 3.10+

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируй `.env.example` в `.env`:

   **Windows (cmd):**
   ```cmd
   copy .env.example .env
   ```

   **Windows (PowerShell) / Linux / macOS:**
   ```bash
   cp .env.example .env
   ```

2. Заполни переменные в `.env`:

   | Переменная | Описание |
   |------------|----------|
   | `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
   | `OPENAI_API_KEY` | Ключ API [OpenAI](https://platform.openai.com/api-keys) |
   | `OPENAI_MODEL` | Модель чата: `gpt-5-mini-2025-08-07`, `gpt-4o-mini`, `gpt-4o` и т.д. |
   | `SORA_MODEL` | Модель для видео: `sora-2`, `sora-2-pro` |
   | `IMAGE_MODEL` | Модель для изображений: `gpt-image-1-mini`, `gpt-image-1`, `gpt-image-1.5` |
   | `MAX_HISTORY_MESSAGES` | Максимум сообщений в истории диалога (по умолчанию: 10) |

## Запуск

```bash
python main.py
```

При запуске в терминал выводится количество режимов, история сообщений и используемая модель.

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и справка |
| `/mode` | Выбор режима (inline-кнопки) |
| `/reset` | Очистить историю диалога |
| `/video` | Генерация видео Sora 2 — ввод промпта, создание 4‑секундного видео |
| `/image` | Генерация изображения (gpt-image-1-mini) — ввод промпта |
| `/cancel` | Отменить текущее действие (выйти из режима видео/изображений) |

## Режимы

Режимы задаются в `prompts.json`. Можно добавлять и редактировать системные промпты. По умолчанию: обычный помощник, разработчик, редактор, учитель, аналитик.

## Функции

- **Память диалога** — история последних N сообщений для каждого чата (JSON)
- **Стоимость запросов** — после каждого ответа выводятся входные/выходные токены и примерная стоимость в ₽ (курс ЦБ РФ)
- **Логирование** — события пишутся в консоль: сообщения, смена режима, генерация видео, ошибки
- **Видео Sora 2** — отдельный поток: `/video` → промпт → ожидание → MP4
- **Изображения** — отдельный поток: `/image` → промпт → PNG 1024×1024

## Структура проекта

| Файл | Назначение |
|------|------------|
| `main.py` | Точка входа, хэндлеры, OpenAI и Sora 2 |
| `config.py` | Конфигурация из `.env` |
| `memory.py` | Хранение истории диалогов (JSON) |
| `utils.py` | Курс ЦБ РФ, расчёт стоимости запросов |
| `prompts.json` | Системные промпты (режимы) |
| `memory.json` | Сохраняемая история (создаётся при работе) |
